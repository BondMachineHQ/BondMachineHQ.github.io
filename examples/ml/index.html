Machine Learning

These examples demonstrate the use of the machine learning capabilities of the BondMachine. The examples use the neuralbond tool, which is part of the BondMachine project. As all the other examples, the code is hosted on the BondMachine Examples Repository. There is also a second repository (bmmltests), which contains tests specifically for the machine learning tool. It contains code to train and test the models, and also to export the models to be used in the BondMachine framework.
