Assembly Reference

This directory contains reference documentation for the BASM assembly language. The instructions listed here are only the 1 on 1 mapping of the machine code instructions that are available in the BondMachine project. BASM supports these instructions and pseudo-instructions as well. The pseudo-instructions are not listed here.

Dynamical Instructions

call fixed_point flopoco linear_quantizer rsets stack

Not all instructions are fully supported by the BondMachine project. Some instructions are in the process of being implemented. The support matrix for the instructions is available here.
